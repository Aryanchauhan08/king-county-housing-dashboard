# King County Housing Dashboard

A professional Streamlit dashboard for analyzing King County housing data.

## Features
*   **Home:** Executive summary and key market metrics.
*   **Univariate Analysis:** Deep dive into individual features.
*   **Bivariate Analysis:** Compare features against price (Scatter, Box Plots).
*   **Multivariate Analysis:** 3D and complex interactions.
*   **Geospatial Intelligence:** Interactive Folium map with dual layers (Price & Living Area).

## Setup

1.  **Install Dependencies:**
    ```bash
    pip install -r requirements.txt
    ```

2.  **Run Application:**
    ```bash
    streamlit run app.py
    ```
